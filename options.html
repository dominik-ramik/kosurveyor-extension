KoSurveyor proxy

Configure which KoboToolbox server this extension proxies requests for.

Default: https://kf.kobotoolbox.org
Change this only if your team uses a self-hosted KoboToolbox instance. No trailing slash.

A browser permission prompt will appear to allow the extension to access the new server. You must accept it for the extension to work. The extension doesn't store or share any data. Its sole purpose is to proxy requests from the KoSurveyor web app to the KoboToolbox API, bypassing the browser's CORS restrictions. You can review the extension's source code on GitHub.
